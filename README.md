## 土木建筑知识问答系统 - 基于知识图谱的专业知识管理平台

**项目简介**

土木建筑知识问答系统是一个基于知识图谱技术的平台，旨在提供专业人士和爱好者快速、准确的土木建筑知识检索和问答服务。该系统利用知识图谱技术有效地组织和管理土木建筑领域的专业知识，并以直观的图谱形式展示和支持高效的查询。

**主要功能**

- **知识图谱展示与查询：** 以图谱形式可视化展示土木建筑知识之间的关联和脉络，并支持高效的图谱查询。
- **数据统计与分析：** 提供丰富的数据统计功能，可根据数据库中的数据生成柱状图等统计数据，帮助用户了解土木建筑知识的分布和热点。
- **数据初始化：** 支持从文件中导入数据到neo4j数据库，确保数据的准确性和完整性。

**技术架构**

- **后端：** Python语言，py2neo库（与neo4j数据库交互）
- **前端：** echarts库（数据统计和知识图谱可视化）

**使用说明**

1. 登录系统并创建账号。
2. 平台提供了多种查询和统计功能，以满足您的不同需求。
3. 数据统计页面提供各种指标的柱状图，帮助您了解土木建筑知识的分布和流行度。
